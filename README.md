# TomatoClock-for-WearOS
一款专为 WearOS 手表打造的极简、纯净、高效的番茄钟应用。采用 **Jetpack Compose** 现代架构开发，完美适配圆形表盘。

✨ 功能亮点
- **极简设计**：无广告、无冗余，只需一键点击，开启专注。
- **AMOLED 优化**：深度纯黑主题（Pure Black），大幅降低手表能耗。
- **动态交互**：专为 WearOS 小屏幕设计的交互逻辑与实时统计功能。
- **开源共创**：这是一个由人类开发者、社区伙伴与 AI 协作诞生的实验性项目。
